## What is Nexus AI?

Nexus AI is a lightweight computer assistant that combines useful system tools, computer information, file utilities and interactive features in one simple application.

It helps users perform common computer tasks faster while keeping everything accessible from one place.

---

## Features

- Computer information
- Hardware and system reports
- File utilities
- Useful system tools
- Interactive Small Talk
- English & Russian support
- Command history
