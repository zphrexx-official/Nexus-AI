# 🧠 About Nexus AI

Nexus AI is a modern command-based computer assistant created to simplify everyday computer tasks.

Instead of searching through numerous operating system utilities, Nexus AI brings useful tools, computer information, reports, diagnostics and interactive features together inside one lightweight application.

The project focuses on:

- simplicity
- speed
- convenience
- portability
- user-friendly interaction

Nexus AI is designed to help both beginners and experienced users access useful computer features much faster.

# ✨ Features

### 🖥 Computer Information

- System information

---

### 📁 File Utilities

- File operations
- Folder utilities

---

### 🧰 System Tools

- Built-in Windows utilities
- Helpful computer commands
- Quick access to useful functions

---

### 💬 Interactive Small Talk

Nexus AI can respond to predefined conversational questions.

Examples:

- Who are you?
- What is your purpose?
- Can you help me?
- Are you alive?
- Why were you created?

---

### 📜 Command History

Every executed command is automatically saved.

The history system allows users to review previously used commands during future sessions.

---

### 🌐 Languages

Supported languages:

- 🇬🇧 English
- 🇷🇺 Russian

The preferred language is selected when Nexus AI starts.

---

### ⚡ Built-in Mini Game

Nexus AI includes a small **Reaction Test** mini game.

Measure your reaction speed directly inside the interactive documentation.

Challenge yourself and compare your best reaction time.
